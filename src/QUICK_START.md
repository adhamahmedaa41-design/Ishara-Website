## 🔄 Switching Views

### **Quick Switch**
Look at the **top center** of your screen:
- Click **"Website"** for the cinematic experience
- Click **"Mobile App"** to see the mobile interface

### **Keyboard Shortcuts** ⌨️
- Press `Shift + W` → Switch to Website
- Press `Shift + M` → Switch to Mobile
- Press `Shift + ?` → Show all shortcuts

1. **Hero Section** 🎬

   - Floating particles in background
   - Click "Explore Technology" to jump to demo

2. **Technology Section** 🔧
   - **Try the Sensor Simulator:**
     - Drag the distance slider
     - Watch colors change (Green → Orange → Red)

4. **Learning Section** 📚
   - Interactive sign language demos
   - Educational quizzes
   - Progress tracking

5. **Hardware Section** 💎
   - 3D product visualization
   - Animated specifications
   - Feature highlights

### **Try These:**

## ⚡ Performance Tips

### **Best Experience:**
- **Browser:** Chrome, Firefox, Safari, Edge (latest)
- **Screen:** 1280x720 or larger
- **Zoom:** 80-100% for mobile view
- **Connection:** Broadband for smooth animations

### **If Animations Lag:**
- Close other tabs
- Disable browser extensions
- Check GPU acceleration enabled
- Try reducing system motion settings

## 🔧 Common Questions

### **Q: Switcher disappeared?**
**A:** Hover your mouse over the top center area - it will slide down!

### **Q: Mobile frame too big/small?**
**A:** Use browser zoom: `Ctrl/Cmd + -` or `Ctrl/Cmd + +`

### **Q: How do I reset everything?**
**A:** Open browser console and run:
```javascript
localStorage.clear();
location.reload();
```

### **Q: Can I use keyboard only?**
**A:** Yes! Press `Shift + ?` to see all shortcuts

### **Q: What about accessibility?**
**A:** Full WCAG compliance, screen reader support, keyboard navigation
